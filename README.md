# 吉本4劇場スケジュール検索

個人用のローカルWebアプリです。吉本公式ページが利用している公開APIから、次の4劇場のスケジュールを取得します。

- ルミネtheよしもと
- 渋谷よしもと漫才劇場
- 神保町よしもと漫才劇場
- YOSHIMOTO ROPPONGI THEATER

## 使い方

このフォルダでローカルサーバーを起動します。

```powershell
npm start
```

または、依存なしで直接起動します。

```powershell
node server.mjs
```

ブラウザで `http://localhost:4173` を開きます。

## iPhoneで使う

GitHub PagesなどのHTTPS URLで公開すると、iPhoneのSafariから使えます。

1. Safariで公開URLを開く
2. 共有ボタンを押す
3. 「ホーム画面に追加」を選ぶ

ホーム画面から起動すると、アプリ風の全画面表示になります。スケジュール取得は公式APIへのオンライン接続が必要です。

## 機能

- 日付を選んで4劇場の公演を一覧表示
- 劇場チェックボックスで対象劇場を絞り込み
- 芸名で出演予定を検索
- 検索期間の指定
- 公式スケジュール、チケット、オンライン、ポスターへのリンク表示
- ホーム画面追加用のPWA設定

取得できる範囲は、公式APIに登録されているスケジュールに依存します。
